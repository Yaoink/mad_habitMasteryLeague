Project Title: Habit Mastery

Description: To track the what habits are done in order to encourage user to develop and maintain one that will be useful to their daily living. 

Features:
- add and edit habit's name and description
- tracks streak

Technology used: flutter and sqlite

Installation: 
1. download this repository
2. in IDE or terminal, navigate to the file path
3. type 'flutter run' as the command for the terminal
   
Usage Guide: 
On homescreen, click add habit and fill out form. Under frequency, type in daily, weekly, or monthly.
